Reduce Method ES6

i- The reduce() method executes a reducer function for array element.
ii- The reduce() method returns a single value: the function's accumulated result.
The reduce() method does not execute the function for empty array elements.
The reduce() method does not change the original array.

Syntax:-

array.reduce(function(total, currentValue, currentIndex, arr), initialValue)

How the code work

1. The input string is split into an array of characters. 2. The reduce() method is applied to the array, starting with an initial value of 0. 3. For each character, the callback function increments or decrements the previous value based on whether it's an opening or closing parenthesis. 4. If the final previous value is 0, the parentheses are balanced, and the function returns true (due to the negation !). 5. If the final previous value is not 0, the parentheses are unbalanced, and the function returns false. In the first example, "(((", the previous value becomes 3 (incremented three times), so the function returns false. In the second example, ")(", the previous value becomes -1 (incremented once and then decremented once), so the function returns false.
